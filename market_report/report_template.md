# 보고서 포맷 (참조용)

`build_report.py` 가 생성하는 `reports/YYYY-MM-DD.md` 의 표준 포맷이다.
실제 렌더링은 코드가 담당하며, 이 문서는 섹션 순서/형식의 단일 기준이다.

```
# 📊 데일리 시장 보고서 — YYYY-MM-DD (요일)

- 보고 생성: YYYY-MM-DD HH:MM KST
- 데이터 기준일: **YYYY-MM-DD** (직전 거래일 종가 기준 · 주말/휴장 시 마지막 거래일)

## 📌 한눈에 보기
(사실 기반 3~5문장 요약)

## 💹 지표 대시보드
> 변동: 일=전일 종가 / 월=1개월 전 / 연=1년 전 대비

### 원자재
| 지표 | 기준일 | 현재가 | 일 | 월 | 연 |
|---|---|---:|---:|---:|---:|
| 금 | YYYY-MM-DD | 2,345.67 $/oz | ▲ +0.50% | ▲ +3.20% | ▲ +18.40% |
...

### 증시
...
### 환율·금리
...
### 암호화폐
...

## 📰 오늘의 뉴스
1. **제목** (매체)
   - 한 줄 요약
   - https://...

## 🧭 오늘의 시장 해설
(수치와 뉴스를 연결한 해설 4~6문장)

---
_본 보고서는 자동 생성되었으며 투자 권유가 아닙니다._
```

## 표시 규칙
- 등락: 양수 `▲`, 음수 `▼`, 변동없음 `—`, 데이터 없음 `N/A`
- 단위: 원자재 $/oz·$/bbl·¢/bu, 증시/달러인덱스 pt, 국채10년 %, 환율 원/엔/$
- 미 국채10년(`^TNX`)은 수익률(값÷10)로 환산해 % 표기
